# CarServiceTracker

CarServiceTracker is an ASP.NET Core MVC web application for managing cars and their service history.

## Features

- Add, edit and delete cars
- Add, edit and delete service records
- Track service date, mileage, price and notes
- Link each service record to a specific car
- Clean and simple Bootstrap UI

## Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- Razor Views

## Project Structure

- Models (3 entities)
    - Car
    - ServiceRecord
    - ServiceType
- Controllers
    - CarsController
    - ServiceRecordsController
- Full CRUD functionality
- Data validation using DataAnnotations

## How to Run the Project

1. Clone the repository
2. Open the solution in Visual Studio
3. Run Update-Database
4. Press F5 to start
